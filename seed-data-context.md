# Acorn Archive 시드 데이터 컨텍스트

> 이 파일은 Acorn Archive 프로젝트의 데모/시드 데이터입니다.  
> DB 시딩, UI 목업, 발표 데모 등 다른 곳에서 자유롭게 활용하세요.

## 설계 의도

- **영화 5개는 사극**: 조선시대·역사 배경 → 한국 전통 미학 취향 신호
- **음악은 K-indie + 전통국악 혼합**: 판소리·가야금·민요 퓨전 → 전통 소리 취향 신호
- **웹툰은 전통/무협/사극 요소 포함**: 동아시아 고전 세계관 취향 신호
- **종합 추천 결과**: 위 세 카테고리를 합산하면 *전통 한국 문화 체험* 쪽으로 추천이 자연스럽게 연결됨
